Map of the gender gap in the Italian universities in 2019

This project allows to visualize an interactive map with data related to Italian universities (Open Data MIUR).
The project uses R and is easily reproducible, being the data and all the resources available online.

Packages Required


The following R packages are required for execution:
  • dplyr
  • leaflet
  • htmltools
  • rgdal
  • sp
  • htmltools
Initial data available on the MIUR website concern the number of academic staff divided by gender, year, and role in the public Italian university for the period 2010-2019.
The data was aggregated using ISTAT's standard Italian Regions names.
